# Kakuu Co., Ltd. Renewal Pattern A

## コンセプト
業務ハブ型トップページ。  
「今すぐ提案に使える情報へ到達する速さ」を最優先に、製品探索・課題別ソリューション・ニュース/セミナーを1画面に集約します。

## ねらい
- 問い合わせ導線の短縮
- 製品検索利用率の向上
- セミナー申込率の改善

## 情報設計の特徴
- ファーストビューに案件クイック起点（検索UI）
- 目的ベース導線（セキュリティ、クラウド、AI、BCP）
- KPIカードでサイトの価値を即時提示
- ニュースとセミナーを並列配置して比較閲覧しやすく設計

## ファイル
- `index.html`: モック本体
- `styles.css`: テーマ/レスポンシブ/アニメーション
